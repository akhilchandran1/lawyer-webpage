# Lawyer Personal Website

Single-page, professional website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Lint

```bash
npm run lint
```

## Deployment

This project is a standard Next.js App Router app and can be deployed to any Node.js hosting platform that supports Next.js (e.g., Vercel, Netlify, or a custom server).
